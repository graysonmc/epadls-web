# EPADLS Scheduler

A Google Apps Script-based scheduling system for EPADLS service management.
## Setup

1. Install dependencies:
  ```bash
  npm install -g @google/clasp
  npm install

Login to clasp:
bashclasp login


Development Workflow
Working with Development Environment
bash# Push changes to development
npm run push:dev

# Pull changes from development
npm run pull:dev

# Open development script in browser
npm run open:dev

# Deploy to development
npm run deploy:dev
Working with Production Environment
bash# Push changes to production
npm run push:prod

# Pull changes from production
npm run pull:prod

# Open production script in browser
npm run open:prod

# Deploy to production
npm run deploy:prod
Other Commands
bash# View logs
npm run logs

# Check status
npm run status

# Backup current version
npm run backup
Project Structure

Code.js - Main Apps Script code
appsscript.json - Apps Script manifest
.clasp-dev.json - Development environment configuration (not in git)
.clasp-prod.json - Production environment configuration (not in git)
